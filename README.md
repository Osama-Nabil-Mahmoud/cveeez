# CVEEEZ Corporate Profile

منصة احترافية لشركة **CVEEEZ** مبنية باستخدام Next.js، TailwindCSS، ودعم كامل للغتين (العربية/الإنجليزية).

## المميزات الحالية
- **دعم اللغات**: تبديل فوري بين العربية (RTL) والإنجليزية (LTR).
- **الوضع الليلي والنهاري**: دعم كامل للـ Dark & Light mode مع حفظ تفضيلات المستخدم.
- **Hero Section**: تصميم عصري ثلاثي الأبعاد يعبر عن خدمات المنصة (بناء CV، بوابة وظائف، تحليل ATS).
- **الخدمات**: عرض خدمات بناء السيرة الذاتية بالذكاء الاصطناعي وبوابة الوظائف بشكل شبكي احترافي.
- **التوافق**: متوافق تماماً مع الهواتف والأجهزة اللوحية (Responsive Design).
- **التواصل والسياسات**: صفحات مخصصة لعرض معلومات الاتصال المباشرة وسياسات الاستخدام والخصوصية.

## التقنيات المستخدمة
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS + ShadCN UI
- **Icons**: Lucide React
- **AI Integration**: Google Genkit (جاهز للتوسعة)
- **Deployment**: جاهز للنشر على Vercel

## تعليمات التشغيل المحلي
1. قم بتثبيت التبعيات: `npm install`
2. ابدأ خادم التطوير: `npm run dev`
3. قم بزيارة: `http://localhost:9002`

## النشر (Deployment)
المشروع جاهز للنشر المباشر على **Vercel**:
1. ارفع الكود على مستودع (Repository) في **GitHub**.
2. اربط المستودع بـ **Vercel**.
3. سيتم الكشف عن إعدادات Next.js تلقائياً وبدء عملية الـ Build.
