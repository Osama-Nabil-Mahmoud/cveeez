# **App Name**: CVeeez Profile

## Core Features:

- Homepage: Displays company information, key features, customer testimonials, and calls to action in AR/EN.
- AI CV Builder Link: Provides a direct link to the AI CV Builder tool, prompting login if the user is not authenticated.
- Job Board Link: Links to the CVEEEZ Job Board, allowing users to search and browse available job listings.
- Policies Page: Presents a comprehensive page with summaries and links to the Terms, Privacy, and Refund policies.
- Contact Form: Allows users to send inquiries via a contact form, optionally integrating with Firestore to store messages or defaulting to mailto:.
- Multi-language Support: Enables content translation between Arabic and English with a language toggle, defaulting to Arabic.
- Theme Toggle: Allows the user to switch between dark and light themes

## Style Guidelines:

- Primary color: Orange (#C65C2B) for a vibrant and professional look.
- Secondary color: Teal (#136A66) to complement the orange and provide balance.
- Background color: Charcoal (#17181F) for a dark-first theme with good contrast.
- Text color: Off-white (#F5F5F5) for readability on dark backgrounds.
- English font: 'Inter', sans-serif, for a modern and clean feel.
- Arabic font: 'Cairo', sans-serif, for a clear and readable Arabic text.
- Use simple and professional icons to represent different services and features.
- Responsive layout adapts to different screen sizes, ensuring a consistent user experience across devices.
- Subtle animations and transitions to enhance user engagement without being distracting.
